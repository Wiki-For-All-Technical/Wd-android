The Organization

The Wikimedia Foundation is the non-profit organization that hosts Wikidata, Wikipedia, and other free knowledge projects. It provides the infrastructure and framework for global collaboration.

Knowledge Base

Wikidata is a free, collaborative, multilingual, secondary database, collecting structured data to support Wikipedia, Wikimedia Commons, and other projects, released under the CC0 license.

App Integration

This Android application is built to interact directly with the Wikidata ecosystem via the official MediaWiki API, utilizing OAuth 2.0 to ensure secure and attributable edits.

Open Principles

Aligned with the mission of free knowledge, the project promotes open data accessibility, allowing anyone to contribute, edit, and utilize structured knowledge globally.
