Executive Summary

Abstract

Bridging the gap between mobile convenience and structured data contribution.

Project Purpose

Enable full-featured viewing and editing of Wikidata directly on Android devices, empowering contributors everywhere.

Current Limitations

The existing mobile web interface offers limited editing capabilities, hindering quick, on-the-go contributions for power users.

Proposed Solution

A native application providing a complete toolset for managing items, properties, statements, references, and qualifiers.

Enhanced Security

Integration of OAuth 2.0 via the Wikimedia API ensures that all user contributions are securely authorized and tracked.

Modern UX

Built with Material Design principles, featuring expandable cards and responsive lists for a high-quality, intuitive user experience.

Data Reliability

Includes a local Activity Log powered by Room Database to track edit history, timestamps, and synchronization status offline.
